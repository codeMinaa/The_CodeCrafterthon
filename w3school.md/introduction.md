# What is Go?

* Go is an open source programming language
* Go is used to create high-performance applications
* Go is a statically typed, compiled and fast language known for its simplicity and efficiency
* Go was founded and developed at Google by Robert Griesemer, Rob Pike, and Ken Thompson in 2007
* Go has similar syntax to C++

## What is Go Used For?
* Web development (server-side)
* Developing network-based programs
* Developing cross-platform enterprise applications
* Cloud-native development

# Why Use Go?
* Go is fun and easy to learn
* Go has fast run time and compilation time
* Go supports concurrency
* Go has memory management
* Go works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc.)

# Go Compared to Python and C++
* Go statically typed, Python is dynamically typed and C++ statically.
* Go is compiled, Python is interpreted and C++ is compiled
* Go is fast, Python is slow in time and C++ is fast also.
